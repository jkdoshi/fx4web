PROJECT: fx4web toolkit demo
----------------------------

This project shows a demo web-application that uses the features provided by
fx4web toolkit.

Please note that this web-application will not compile/run out-of-the-box.
You will have to copy the required libraries as mentioned in
WebContent/WEB-INF/lib/libraries.txt.

The main site for this project is http://code.google.com/p/fx4web.
