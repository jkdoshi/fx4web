fx4web-demo

Welcome to fx4web library demo application. This web-application demonstrates the features of the fx4web library. The following are the available demos.

  • sloshbucket: Here, we show how a regular select-many listbox is turned into a sloshbucket (a pair of listboxes with a pair of buttons that allow moving items between the two listboxes).
  • sloshbucket: Here, we show how a regular select-many listbox is turned into a sloshbucket (a pair of listboxes with a pair of buttons that allow moving items between the two listboxes).
  • tiles-demo: Here, we show how tiles can be integrated with JSF (using view-handler definition in faces-config.xml).
  • conversations demo: Here, we show how multiple child windows can maintain their own conversation context.
  • multi-file-selection applet demo: Here, we show how a signed applet can allow your web-page to select multiple files (possibly for file-upload) using a single file-open dialog.
