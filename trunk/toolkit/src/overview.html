fx4web toolkit

fx4web toolkit provides utilities for web-programming in Java. Aside of miscellenious stuff, the main features are ...

  1. Implementation of HTTP "conversation" scope (akin to "request" and "session" scope for attributes/beans).
  2. A JSF PhaseListener that places a "floating" window on all your pages containing JSF messages that are hyperlinked to the components the message pertains to.
